This document is the API specification for the Hadoop integration for DJL.

You can use this library to load models from Hadoop HDFS file system: 

    Criteria<Image, Classifications> criteria =
        Criteria.builder()
                .optApplication(Application.CV.IMAGE_CLASSIFICATION)
                .setTypes(Image.class, Classifications.class)
                .optModelUrls("hdfs://localhost/var/resnet/resnet18.zip")
                .optArtifactId("ai.djl.localmodelzoo:resnet18")
                .optProgress(new ProgressBar())
                .build();

    ZooModel<Image, Classifications> model = criteria.loadModel();
